# 🚀 Arc Agent Hub v8.1

> **Premium AI Agent Operations Platform for Arc Testnet**

Arc Agent Hub v8.1 is a production-ready, open-source developer platform
showcasing Circle AppKit, ERC-8004 AI Identity, ERC-8183 Agent Commerce,
Universal Wallet, Payments, Circle CCTP Bridge, Universal Token Swap,
Trust Center and Mission Control.

# 🧩 Platform Modules

Realtime dashboard, analytics, activity and portfolio.

Balances, assets, history and wallet connectivity.

## Bridge

Native Circle CCTP bridge workflow.

## Swap

# 🚀 Getting Started

``` bash
git clone https://github.com/Jaehaerysp/arc-agent-hub-v8.git
cd arc-agent-hub-v8
npm install
npm run dev
```

Build:

``` bash
npm run build
npm run preview
```
